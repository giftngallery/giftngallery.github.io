💌 Resend.com Email Setup
1. Sign up free at resend.com (100 emails/day free)
2. Get your API Key from Resend dashboard
3. Verify your sender domain or use onboarding@resend.dev for testing
4. Deploy the Edge Function (instructions below)
5. Save your settings here — you're done!
Get from resend.com › API Keys
Must be verified in Resend. For testing use: onboarding@resend.dev
Using resend.dev sandbox sender — it can only deliver to the email address you signed up to Resend with. Order + customer details will be emailed here instead of to the customer.
🚀 Supabase Edge Function (deploy once)
// supabase/functions/send-order-email/index.ts // IMPORTANT: your site is on a different origin (e.g. GitHub Pages) than // *.supabase.co, so the browser needs explicit CORS headers on every // response, including the OPTIONS preflight — otherwise fetch() fails // silently with no visible error. const corsHeaders = { "Access-Control-Allow-Origin": "*", "Access-Control-Allow-Headers": "authorization, x-client-info, apikey, content-type", "Access-Control-Allow-Methods": "POST, OPTIONS" }; Deno.serve(async (req) => { if (req.method === "OPTIONS") { return new Response(null, { headers: corsHeaders }); } try { const { to, subject, html, resend_api_key, from_email } = await req.json(); const res = await fetch("https://api.resend.com/emails", { method: "POST", headers: { "Authorization": "Bearer "+resend_api_key, "Content-Type": "application/json" }, body: JSON.stringify({ from: from_email, to, subject, html }) }); const data = await res.json(); return new Response(JSON.stringify(data), { status: res.status, headers: { ...corsHeaders, "Content-Type": "application/json" } }); } catch (err) { return new Response(JSON.stringify({ error: String(err) }), { status: 500, headers: { ...corsHeaders, "Content-Type": "application/json" } }); } });
Deploy: supabase functions deploy send-order-email --no-verify-jwt
If you already deployed the old version, you must redeploy with this updated code — the CORS headers are what was silently blocking emails.
